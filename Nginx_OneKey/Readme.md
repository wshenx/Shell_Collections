# Nginx_OneKey
Install nginx onekey.
## Usage:
```shell
wget --no-check-certificate https://github.com/Char1sma/Shell_Collections/raw/master/Nginx_OneKey/onekey.sh && bash onekey.sh install | tee -a nginx_onekey.log
```
